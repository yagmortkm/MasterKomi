# MasterKomi
<h2>Описание</h2>
Это единый агрегатор и витрина мастер-классов. Пользователь может зарегестрировать аккаунт, войти в личный кабинет, записаться на мастеркласс, посмотреть календарь событий. Тажке, возможна фильтрация по направлениям (творчество, спорт, IT), городам и сортировка по времени.
<h2>Техническое задание</h2>
Создание бд с таблицами: users, events, cities, sponsors
Создание форм для регистрации, авторизации, записи на мероприятие.
Создание обработчиков.
